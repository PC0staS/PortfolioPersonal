# CafeShop ☕

Sistema completo para la gestión de productos de café. Incluye backend con API REST segura, panel de administración con autenticación JWT y un frontend responsive con transiciones suaves.

## ¿Qué hice?

- API **FastAPI** con modelos SQLAlchemy y validación **Pydantic**.
- Autenticación y autorización con **JWT** (PyJWT).
- **CRUD de productos** y gestión de múltiples imágenes por producto.
- **Panel admin** moderno y protegido.
- Frontend **Next.js 15 (React 19, TypeScript)** + **Tailwind CSS** con View Transitions.
- Despliegue orquestado con **Docker Compose** (PostgreSQL + backend + frontend).

---

## Características clave

- CRUD completo (crear, leer, actualizar, eliminar).
- Subida/gestión de imágenes por producto con previsualización.
- Diseño responsive y accesible.
- Seguridad: JWT, validación de datos, CORS, saneado de inputs.
- Scripts de verificación de entorno y guía de instalación.

---

## Arquitectura

- Backend (Python / FastAPI):
  - Endpoints REST, esquemas Pydantic, ORM SQLAlchemy.
  - BD: **PostgreSQL 16**.
  - Docker: imágenes del backend y orquestación con `docker-compose`.
- Frontend (Next.js 15 / React 19 / TS):
  - App Router, componentes UI y páginas de productos/admin.
  - Estilos con **Tailwind CSS** y View Transitions.

### Endpoints principales (resumen)
- Públicos:
  - `GET /products` — listar productos (paginado)
  - `GET /products/{id}` — detalle de producto
  - `GET /products/{id}/images` — imágenes del producto
- Administración (JWT requerido):
  - `POST /admin/auth/register`, `POST /admin/auth/login`, `POST /admin/auth/refresh`
  - `POST /admin/products`, `PUT /admin/products/{id}`, `DELETE /admin/products/{id}`
  - `POST /admin/products/full` — crear producto con imágenes

## Instalación y ejecución

- Opción Docker (recomendada):
  ```bash
  git clone https://github.com/PC0staS/CafeShop.git
  cd CafeShop
  cd backend/code && cp .env.example .env # configura DB_URL, JWT_SECRET, etc.
  docker-compose up --build
  ```
- Frontend en otra terminal:
  ```bash
  cd frontend
  npm install --legacy-peer-deps
  npm run dev
  ```
- Acceso:
  - Frontend: http://localhost:3000
  - Admin: http://localhost:3000/admin
  - API: http://localhost:8000 — Docs: http://localhost:8000/docs
