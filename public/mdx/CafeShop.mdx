# CafeShop ☕

Sistema completo de e-commerce para café con API FastAPI, panel admin JWT y frontend Next.js responsive.

## En una frase
E-commerce completo: API FastAPI + Next.js con CRUD de productos, imágenes y autenticación JWT.

## Stack
- **Backend:** FastAPI, SQLAlchemy, Pydantic, JWT, PostgreSQL
- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS
- **Deploy:** Docker Compose

## Highlights
- **API robusta:** FastAPI con validación Pydantic y ORM SQLAlchemy
- **Autenticación JWT:** Login, registro, refresh tokens
- **CRUD completo:** Productos con múltiples imágenes y previsualización
- **Panel admin:** Protegido con JWT, diseño moderno
- **Frontend moderno:** Next.js 15 con View Transitions y Tailwind
- **Deploy fácil:** Docker Compose con PostgreSQL

## Arquitectura

### Backend (FastAPI)
- Endpoints REST con esquemas Pydantic
- PostgreSQL 16 con SQLAlchemy ORM
- Seguridad: JWT, validación, CORS, saneado de inputs

### Frontend (Next.js 15)
- App Router con React 19 y TypeScript
- Tailwind CSS con View Transitions
- Componentes UI y páginas productos/admin

## API endpoints
- Públicos: `GET /products`, `GET /products/{id}`, `GET /products/{id}/images`
- Admin (JWT): `POST /admin/auth/login`, `POST /admin/products`, `PUT /admin/products/{id}`, `DELETE /admin/products/{id}`

## Instalación
```bash
# Backend
cd backend/code && cp .env.example .env
docker-compose up --build

# Frontend
cd frontend && npm install --legacy-peer-deps
npm run dev

# Acceso
# Frontend: http://localhost:3000
# Admin: http://localhost:3000/admin  
# API: http://localhost:8000
```
