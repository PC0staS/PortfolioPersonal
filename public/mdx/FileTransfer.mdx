# FileTransfer - Servidor de Archivos 📁

Plataforma web moderna para compartir archivos en red local con diseño glassmorphism y subidas avanzadas.

## En una frase
Comparte archivos fácilmente con drag & drop, subidas por chunks y enlaces directos.

## Stack
- **Backend:** Flask, SQLite, Python
- **Frontend:** JavaScript vanilla, Bootstrap
- **Deploy:** Docker Compose, Cloudflare Tunnel

## Highlights
- **Interfaz moderna:** Glassmorphism con gradientes y efectos visuales
- **Subidas avanzadas:** Drag & drop, múltiples archivos, progreso en tiempo real
- **Sistema de chunks:** Subidas resumibles para archivos grandes (>200MB)
- **Enlaces directos:** Descargas públicas sin autenticación
- **Gestión de usuarios:** Sistema de aprobación manual
- **Dockerizado:** Despliegue fácil con Docker Compose

## Funcionalidades clave

### 🔐 Autenticación
- Registro con aprobación manual
- Estados: pendiente, activo, rechazado
- Sesiones seguras con cookies HttpOnly

### 📤 Subidas Avanzadas
- **Drag & Drop:** Interfaz intuitiva
- **Múltiples archivos:** Subida simultánea con progreso individual
- **Chunks resumibles:** Para archivos grandes con reanudación
- **Progreso en tiempo real:** Barras con velocidad y ETA

### 📁 Gestión
- Organización por usuario
- Expiración automática (5 días)
- Metadatos completos
- Iconos por tipo de archivo

## API
- `POST /api/upload_progress` - Subida con progreso
- `POST /api/chunk/init` - Inicializar subida por chunks  
- `POST /api/chunk/upload` - Subir chunk individual
- `GET /download/<filename>` - Descarga pública

## Deploy con Docker
```bash
# .env
HOST_IP=192.168.1.100
DB_VOLUME=/ruta/absoluta/db
UPLOADS_VOLUME=/ruta/absoluta/uploads

# Ejecutar
docker-compose up -d
```
