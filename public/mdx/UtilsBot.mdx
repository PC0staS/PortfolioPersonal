# UtilsBot - Bot de Discord 🤖

Bot multifuncional para Discord con 30+ comandos, administración remota de Raspberry Pi y utilidades avanzadas.

## En una frase
Automatiza tareas de Discord: red, sistema, multimedia, productividad y criptografía.

## Stack
- Python, discord.py, asyncio, aiohttp
- APIs: MyMemory, Open-Meteo, DictionaryAPI
- psutil, ffmpeg, pypdf, cryptography

## Comandos destacados

### 🌐 Red
- `/ping <ip>` `/speedtest` `/netdevices` `/whois <dominio>`

### 🛠️ Sistema
- `/stats` `/reboot` `/shutdown` `/execute <comando>` `/vpnstatus`

### 📁 Multimedia
- `/mergepdf` `/mergevid` `/screenshotweb <url>` `/qr <url>`

### 💡 Productividad
- `/remind <min> <mensaje>` `/habit <min> <mensaje>`
- `/translate <texto>` `/weather <lugar>`

### 🔐 Criptografía
- `/encrypt <mensaje>` `/hash <mensaje>` `/passw <chars>`

## Funcionalidades avanzadas
- **Sistema de hábitos:** Recordatorios recurrentes cancelables
- **Integración Nextcloud:** Guarda archivos generados automáticamente
- **Manejo robusto:** Timeouts, fallbacks, errores graceful

## Deploy
Ejecuta como servicio systemd en Raspberry Pi con auto-restart.

## Casos de uso
Administración remota, herramientas de red, productividad, multimedia, utilidades, información.
