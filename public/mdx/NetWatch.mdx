# NetWatch - Monitor de Red 🌐📊

**NetWatch** es una elegante herramienta de monitoreo de red que desarrollé para supervisar el tráfico de interfaces de red en tiempo real, con tanto interfaz de consola como interfaz web moderna.

## ¿Qué hice?

Desarrollé una aplicación completa de monitoreo de red utilizando **Python**, **Flask**, **Socket.IO** y **Chart.js**. El proyecto incluye dos modos de funcionamiento: una interfaz de consola colorida con gráficos ASCII y una interfaz web profesional con gráficos interactivos.

### Características principales:
- **Doble interfaz**: Consola con gráficos ASCII y web con visualización moderna
- **Monitoreo en tiempo real**: Actualización automática de estadísticas cada segundo
- **Diseño profesional**: Paleta de colores pastel y diseño responsivo
- **API REST**: Endpoints para integración con otros sistemas
- **Multiplataforma**: Compatible con Windows, Linux y macOS

---

## Tecnologías utilizadas

### Backend:
- **Python 3.7+**: Lógica principal de la aplicación
- **psutil**: Obtención de estadísticas de red del sistema
- **Flask & Flask-SocketIO**: Servidor web y comunicación en tiempo real
- **WebSocket**: Para actualizaciones en tiempo real sin recargar la página

### Frontend:
- **HTML5 & CSS3**: Estructura y estilos modernos
- **JavaScript (Vanilla)**: Lógica del cliente y manejo de datos en tiempo real
- **Chart.js**: Gráficos interactivos y visualizaciones
- **Bootstrap 5**: Componentes UI y diseño responsivo

---

## Interfaz de consola

La interfaz de consola incluye:
- Selección interactiva de interfaces de red
- Gráficos ASCII coloridos en tiempo real
- Estadísticas de velocidad actual y tráfico acumulado
- Colores diferenciados para envío (azul) y recepción (verde)

```bash
# Ejecutar en modo consola
netwatch

# Monitorear interfaz específica
netwatch -i eth0

# Monitorear todas las interfaces
netwatch --all
```

---

## Interfaz web profesional

La interfaz web moderna incluye:
- **Gráficos interactivos**: Visualización en tiempo real con Chart.js
- **Diseño responsivo**: Adaptable a móviles, tablets y desktop
- **Paleta pastel**: Colores suaves y elegantes
- **Sin datos simulados**: Muestra estado desconectado cuando no hay datos reales
- **Control intuitivo**: Selección fácil de interfaces y controles de monitoreo

```bash
# Ejecutar interfaz web (localhost)
netwatch-web

# Ejecutar con acceso de red
netwatch-web --network

# Puerto personalizado
netwatch-web --network --port 8080
```

---

## API REST

El proyecto incluye endpoints REST para integración:

- `GET /api/interfaces` - Lista interfaces disponibles
- `GET /api/stats` - Estadísticas actuales de red
- `POST /api/start_monitoring` - Iniciar monitoreo de interfaces
- `POST /api/stop_monitoring` - Detener monitoreo
