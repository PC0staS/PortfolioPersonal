# BlogReactNative 📱

App de blog full‑stack multiplataforma con React Native/Expo y Flask: autenticación JWT, CRUD completo y cámara integrada.

## En una frase
Blog completo con frontend móvil/web en Expo y backend Flask con autenticación JWT.

## Stack
- **Frontend:** React Native/Expo, TypeScript, Expo Router, Reanimated
- **Backend:** Flask, JWT, SQLite, Docker
- **Features:** Cámara/galería, AsyncStorage, navegación file‑based

## Highlights
- **Autenticación completa:** Registro, login, verificación de token
- **CRUD de artículos:** Con imagen base64, autor y fecha
- **Integración cámara:** Expo Image Picker con previsualización
- **Búsqueda en tiempo real:** Con estados de carga y manejo de errores
- **Multiplataforma:** iOS, Android y Web
- **Arquitectura limpia:** File‑based routing + API REST

## API endpoints
- Auth: `POST /api/auth/signup`, `POST /api/auth/login`, `GET /api/auth/check`
- Posts: `GET /api/posts`, `GET /api/posts/{id}`, `POST /api/posts`, `PUT /api/posts/{id}`, `DELETE /api/posts/{id}`

## Ejemplo de uso
```ts
await fetch(`${BACKEND_URL}/api/posts`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Authorization: `Bearer ${token}`,
  },
  body: JSON.stringify({
    title: 'Mi nuevo artículo',
    content: 'Contenido...',
    thumbnail_base64: 'data:image/jpeg;base64,...',
    author: 'Autor',
  }),
});
```

## Deploy
- Backend: Docker Compose o entorno virtual
- Frontend: `npx expo start` para desarrollo, builds para producción
- Config: `BACKEND_URL` en frontend, `.env` en backend
